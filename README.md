# Digital Household Security

A practical household guide to digital security, safety, and privacy.

The public site is served from the `public/` folder:

- `public/index.html` - the guide itself.
- `public/_headers` - Cloudflare security headers.
- `public/robots.txt` - open indexing policy.

Project notes for future editing sessions live outside `public/` so they are not deployed as website files.

Cloudflare Workers Git deployments use `wrangler.jsonc`, with `public/` configured as the static assets directory.
